# 常见项目运行手册

## 外部 API 接入排查顺序

1. key 是否存在
2. endpoint 是否正确
3. 本机 DNS 是否正常
4. 本机到目标 443 是否可达
5. 进程是否真正走代理
6. 结果是否入库
7. 前端是否正确展示

## 代理排查顺序

1. 代理客户端是否在运行
2. HTTP / SOCKS 端口分别是多少
3. 系统代理是否真的写入
4. Node 进程是否继承到代理
5. 如果没有继承，是否要在代码里显式指定 agent

## UI 偏差排查顺序

1. 先确认用户真实业务目标
2. 再确认页面是否暴露了内部实现概念
3. 再确认结果字段是否与业务目标一致
4. 最后才去调视觉细节

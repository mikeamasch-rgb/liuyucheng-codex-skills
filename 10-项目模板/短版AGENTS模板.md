# 短版 AGENTS 模板

以下模板用于新项目的短版 `AGENTS.md`。

```md
# AGENTS.md

## 项目目标

- 用一句话说明项目核心目标
- 用 3-5 行说明主用户流程

## 常驻规则

1. 解决真实问题
2. 只做当前主流程
3. 先计划后编码
4. 不加 mock/demo 数据，除非用户明确要求
5. 每次改动必须验证
6. 用户可见错误保持简短，不暴露技术细节

## 路由

- UI 相关 -> 指向某个 UI 文档
- API 相关 -> 指向某个 API 文档
- DB 相关 -> 指向某个 DB 文档

## 最小验证

- build 通过
- 变更页面可打开
- 主流程可测试
```

## 模板原则

1. 保持短
2. 保持项目化
3. 不在这里塞专项细则

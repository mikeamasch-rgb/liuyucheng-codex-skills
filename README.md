# liuyucheng-codex-skills

个人长期复用的 Codex 规则库与工程 skills 模板。

## 这个仓库解决什么问题

1. 每个项目都重复写长总则
2. 规则太长，常驻上下文浪费 token
3. 做项目时经常重复踩坑
4. 换电脑后不好恢复相同工作方式

## 仓库结构

- `00-核心总则`
- `10-项目模板`
- `20-界面规则`
- `30-后端与接口`
- `40-数据库`
- `50-验证与交付`
- `60-依赖与选型`
- `70-skills设计`
- `80-runbooks`

## 使用方式

不要把这个仓库整体塞进每个项目的常驻规则里。

正确方式是：

1. 项目只保留短版 `AGENTS.md`
2. `AGENTS.md` 只路由到本仓库
3. 按需读取对应文档

## 适合谁

适合：

- 需要长期用 Codex 做项目的人
- 想把规则、技能模板、工程方法沉淀下来的人
- 经常换项目、换电脑、换工作目录的人
